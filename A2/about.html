About This Project

This website is a creative representation of the classic fairy tale "Rapunzel". It includes details about the story, characters, and a magical image gallery that brings the tale to life.

Created as a web design project using HTML and styled beautifully with Tailwind CSS, the goal was to showcase the timeless charm of fairy tales through a clean, modern design.

Each page has been carefully designed to reflect the enchanting world of Rapunzel, appealing especially to those who love fantasy and storytelling.
